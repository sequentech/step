To generate a report you need to select the Election Event in the Admin Portal 
and follow the steps.

1. Go to the tab `REPORTS` and click the button `ADD`.
2. Select the type (see special notes to consider for specific types below).
3. Optionally select Election for reporting only an specific election.
4. Optionally choose a template if you have created one earlier.(See more about 
   creating templates below).

### Creating a template

You can add or customize html templates to add images, links or change texts in 
te reports. 


1. In the Admin Portal´s left panel, below the Election events and below `Settings` 
   click on `Templates`.
2. Give a template name and alias and choose the type.
3. In `Choose methods` enable `Document` and the default html document will be shown,
   you can modify this template that will be rendered to PDF later on.
4. Optional: If the template type is used for communications to the user you can
   modify it by enabling `Email` or `SMS`.
5. After doing modifications click on Save.

The system will use created template when generating a document of that type automatically  
i.e. during the tally proccess or exporting data.
Don´t forget that you can use this template when manually [creating reports](/docs/docusaurus/docs/admin_portal/Tutorials/admin_portal_tutorials_create-reports.md) 
in the Reports tab.

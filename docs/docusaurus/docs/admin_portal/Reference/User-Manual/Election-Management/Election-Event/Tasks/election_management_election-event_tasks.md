Monitor the execution of previous tasks and access detailed information about system actions. This feature provides insights into task progress, system behavior, and operational history, ensuring transparency and traceability.

### View a Task

- Select the **eye icon** in the Actions column.
- View detailed information about this specific task.
- Select **Back** to return to the table view.

## Frontend Services

### Voting Portal
- **Role**: User interface for voters to cast their vote.
- **Path**: `step/packages/voting-portal`
- **Technologies**: React, Typescript, GraphQL
- **Key Directories**:
  - `components/`, `hooks/`, `queries/`, `screens/`, `services/`

---

### Admin Portal
- **Role**: Interface for managing and configuring elections.
- **Path**: `step/packages/admin-portal`
- **Technologies**: React, Typescript, GraphQL
- **Key Directories**:
  - `components/`, `screens/`, `services/`, `resources/`

---

### Ballot Verifier
- **Role**: Enables voters to audit and verify their cast ballots.
- **Path**: `step/packages/ballot-verifier`
- **Technologies**: Javascript, Typescript, GraphQL

## Infrastructure

### Cloud & DevOps
- **AWS Services**:
  - SES (Email)
  - SNS (SMS)
  - S3 (Storage)
  - EC2 (Compute)
  - EKS (Kubernetes)
  - Aurora PostgreSQL (Database)
- **Provisioning**:
  - Terraform (IaC)
  - Docker (Containers)
  - Kubernetes (Orchestration)

---

### Security & Monitoring
- **Wazuh**: Threat detection and compliance
- **Prometheus**: System metrics
- **Grafana**: Dashboard visualization
- **Kibana**: Log analysis
- **Vault**: Secrets and credential storage
- **AWS Secret Manager**: Centralized credentials
- **Cloudflare**: DDoS protection, CDN
- **NGINX**: Load balancing, reverse proxy

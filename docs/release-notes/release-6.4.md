# Release 6.4

### Migration to add permissions to keycloak realm

We've added some permissions that need to be added to the keycloak realm:

    - ee-voters-columns
    - ee-voters-filters
    - ee-voters-logs
    - voter-import
    - voter-export
    - voter-delete
    - voter-manually-verify
    - voter-change-password
